# 🤖 AI智能考试系统 - 云帆增强版

基于**云帆培训考试系统开源版**深度改进，集成**人工智能功能**的下一代智能考试平台。

> **感谢原作者**: 本项目基于[云帆培训考试系统](https://gitee.com/wells2333/yf_exam)开源版本开发，感谢云帆团队提供的优秀基础架构！🙏

## 🆕 AI增强功能

### 🎯 **智能题目管理**
- **AI题目提取**: 上传Word/PDF文档，AI自动识别并提取题目
- **智能图文识别**: 支持图片和文字混合文档的智能解析
- **题目类型识别**: 自动识别单选、多选、判断、简答、填空题
- **答案解析生成**: AI自动生成题目解析和知识点标签

### 📚 **知识大纲管理**
- **大纲文档上传**: 支持Word/PDF格式的知识大纲文档上传
- **AI结构解析**: 智能识别文档中的知识点层次结构
- **自动编码生成**: 为知识点自动生成标准化编码
- **题目大纲关联**: AI自动识别题目对应的知识大纲

### 🧠 **AI判分系统**
- **简答题智能判分**: AI自动评分简答题，提供详细反馈
- **整体测试分析**: 智能分析学生整体表现，给出学习建议
- **知识点掌握分析**: 基于AI分析提供个性化学习建议

## 🛠 技术架构

### **核心技术栈**
- **Backend**: Spring Boot + MyBatis Plus + Shiro + MySQL
- **Frontend**: Vue.js + Element UI
- **AI Engine**: Qwen3-32B (千问大模型)
- **Document Processing**: Python + FastAPI
- **Architecture**: KISS原则 (Keep It Simple And Stupid)

### **系统架构**

## 🚀 快速开始

### **环境要求**
```bash
# 基础环境
JDK 1.8+
MySQL 5.7+
Node.js 14+
Python 3.8+

# AI服务
Qwen3-32B 大模型服务 ()
```

### **安装步骤**

1. **数据库初始化**
```sql
-- 导入基础数据
mysql> source docs/安装资源/数据库脚本.sql
-- 导入AI功能表
mysql> source outline_tables.sql
```

2. **后端启动**
```bash
cd exam-api
mvn clean package
java -jar target/exam-api.jar
# 访问: http://localhost:8101
```

3. **前端启动**
```bash
cd exam-vue
npm install
npm run dev
# 访问: http://localhost:8080
```

4. **文档处理服务启动**
```bash
cd dist  # Python服务目录
python app.py
# 服务: http://localhost:8003
```

## 📋 AI功能使用指南

### **📤 智能题目上传**
1. 登录管理员账号
2. 进入 `题库管理` → `题目管理`
3. 点击 `AI上传` 按钮
4. 上传Word/PDF文档
5. 系统自动解析并导入题目

### **📚 知识大纲管理**
1. 进入 `知识大纲` 模块
2. 点击 `上传大纲文档` 按钮
3. 选择学科和年级
4. 上传包含知识点的Word/PDF文档
5. AI自动解析知识结构并入库

### **🎯 AI识别功能**
1. 在 `知识大纲` → `AI识别` 页面
2. 选择学科和年级
3. 点击 `开始批量识别`
4. 系统自动为题目匹配对应知识点

## 🎯 架构设计理念

### **KISS原则 (Keep It Simple And Stupid)**
- ✅ **简单直接**: 避免过度设计，采用最直接的实现方案
- ✅ **模块清晰**: 明确的功能边界和职责分离
- ✅ **易于维护**: 代码结构清晰，便于后续开发和维护

### **AI集成策略**
- **直接调用**: Java后端直接调用AI服务，避免复杂的中间层
- **专业分工**: Python专门处理文档解析，Java处理业务逻辑
- **错误容错**: AI服务不可用时提供Mock数据保证系统可用性
